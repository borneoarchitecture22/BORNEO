Configurazione GitHub

Per pubblicare le modifiche sul sito live, l'editor ha bisogno di accedere al tuo repository GitHub dove è ospitato il sito. Servono solo due informazioni.

Formato: nomeutente/nome-repository — es. antoniodepaola/borneo
Vai su github.com/settings/tokens/new → seleziona repo → genera → incolla qui.
Il token viene salvato solo nel tuo browser.
